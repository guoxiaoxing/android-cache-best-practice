# LeetCode in Java

>本系列文章整理总结LeetCode上的问题解决方案，开发语言是Java。


拿到问题后，我们一般会有以下思路：

- 排序算法
- 二分查找
- 双指针
- 位处理
- 回溯算法
- 动态规划

针对问题提出的方案应该满足以下条件：

- 规范性：书写清晰、布局清晰与命名合理。
- 完整性：完成基本功能，考虑边界条件与做好错误处理。
- 鲁棒性：采用防御式编程，处理无效的输入。
